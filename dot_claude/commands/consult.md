# 技術諮詢

請針對技術選型或架構決策提供專業建議,包含多種方案比較。

## 諮詢流程:

1. **理解需求**
   - 技術選型的關鍵需求
   - 技術限制
   - 專案背景

2. **提供方案**
   - ⚡ 快速方案: 最簡單、快速上線
   - 🛡️ 穩健方案: 平衡、推薦使用
   - 🌟 理想方案: 最佳、長期發展

3. **比較分析**
   - 開發時間
   - 學習曲線
   - 擴展性
   - 效能
   - 維護性

4. **具體建議**
   - 基於專案情況推薦
   - 說明推薦理由

- {技術限制}
- {專案背景}
